Ember-Tutorial-cn
=================

Translate Vic Ramon's Ember Tutorial to Chinese. 原文链接:http://ember.vicramon.com/

**说明**

文章在翻译过程中,所有相关的术语都基本保持英文没有翻译,另外,原文中的代码有CoffeeScript和JavaScript两个版本,在翻译过程中仅有使用JavaScript,如果要查看CoffeeScript的代码,可以查看原文.
